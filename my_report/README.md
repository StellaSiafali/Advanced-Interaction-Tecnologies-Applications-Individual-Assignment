## 1. Video Capture :camera:
  
Για την εκτέλεση του πρώτου παραδοτέου εργάστηκα ως εξής: αρχικά συνδέθηκα στο λογαριασμό μου στο GitHub, διάβασα ώστε να κατανοήσω πλήρως τα ζητούμενα και συνέχισα στα στάδια προκειμένου να υλοποιήσει το παραδοτέο.
